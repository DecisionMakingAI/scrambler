Introduction

The process of calibrating pinhole cameras has bee around for a long. OpenCV has a set of routines to perform the mathematics for this. There are several tutorial articals and videos on doing this. Two very good articles on this are:

  • OpenCV Tutorials
  • Satya Mallick's Camera Calibration using OpenCV
  • Another Python

There are three basic steps to this process:

  1. Acquire a set of checkerboard or circle test pattern images from the camera to calibrate.
  2. Run the camera calibration program and save coefficients to be used for reconstruction.
  3. Develop setup code the properly correct the image. This code will be inserted in the application that is run when the image is to be corrected.

Acquiring Calibration Images

Print a copy of the checkerboard and/or circles test images. These images are downloadable from Checkerboard and Circles. They are also stored in the development directory as pattern.png and acircles_pattern.png.
